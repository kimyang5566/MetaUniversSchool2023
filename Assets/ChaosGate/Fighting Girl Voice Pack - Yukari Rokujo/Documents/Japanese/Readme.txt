 # 音圧未調整データに関して
-------------------------------------------------------------------------------------------------------------------------------------
本製品にはすぐにご利用いただけるよう音圧調整した「調整済みデータ」の他に、ボイスカットのみ行なった「未調整データ」が同封されています。
通常は「調整済みデータ」をご利用頂ければ良いですが、ご自身で音圧を調整したい場合は未調整データをご使用ください。
